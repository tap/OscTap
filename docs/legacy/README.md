and reference. They predate the OscTap rebrand and the current build/CI, so they
are **historical, not current** — for up-to-date information use the top-level
[`README.md`](../../README.md), [`ROADMAP.md`](../../ROADMAP.md), and the guides in
[`docs/`](../).
